The requested file couldn't be loaded from the SPIFFS!

The requested file doesn't exist on the SPIFFS(Serial Peripheral Interface Flash File System).
This either means the SPIFFS partition couldn't be loaded, or the file is missing.
Try running "pio run -t uploadfs -e esp32dev" to flash the files to the ESP32 over USB
Or "pio run -t uploadfs -e esp32dev_ota" to upload the files over WiFi.
